# Tarea1-1-1300-845

En esta tarea debe utilizar los conocimientos adquiridos con RESTful y Swagger. Se debe utilizar la herramienta de desarrollo **Visual Studio 2022**, Lenguaje de programación **C#** y **Swagger**. El formato de los datos que viajarán en el servicio será Json. Esta tarea no tendrá páginas web que lo consuman . **El proyecto debe estar implementado en .Net 8.**

# Requerimientos

Cree un **servicio Restful** en el que va almacenar **Tipos de Especies Marinas** con los siguiente campos.

## Entidades
### Tipos de Especies Marinas
Pueden ser uno de estos 3 tipos
* Organismos Planctónicos: Flotan en la superficie, como el fitoplancton (algas microscópicas) y el zooplancton (animales microscópicos).
* Organismos Nectónicos: Viven en aguas libres, como peces, ballenas, calamares y tiburones.
* Organismos Bentónicos: Viven en el fondo marino, como algas, moluscos, corales y estrellas de mar.

En el mismo servicio se podrán agregar a cada tipo de especie, los datos de las especies marinas:
* Nombre de especie
* Si es de agua dulce o agua salada
* Si tiene aletas o no
* Si puede respirar fuera del agua o no
* Peso en kilos, que usualmente tiene
